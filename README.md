npslider für Contao Open Source CMS
===================================

npslider ist ein jQuery Slider für sämtliche Inhaltselemente auf Basis von bxSlider (http://www.bxslider.com).
Er bietet eine Reihe von Einstellungsmöglichkeiten und Animationen (u.a. Regulierbare Achsen, Ticker-Modus).

Features
--------

* Große Anzahl an Optionen, konfigurierbar im Backend.
* Es können mehrere Slider pro Seite eingesetzt werden.
* Einfache Installation
* Regulierbare Achsen
* Ticker Modus
* Callback Funktionen
* Anpassung der Anzahl gleichzeitiger angezeigter und bewegter Slides im BE möglich


Installation
------------

* Auswahl Elementtyp "Slider"
* Start-Element einfügen
* Größe des Sliders definieren (wichtig)
* Inhaltselemente einfügen (Gleiche Größe wie Slider)
* Stop-Element einfügen
* Im Seitenlayout "jquery" aktivieren (wenn nicht bereits getan)
